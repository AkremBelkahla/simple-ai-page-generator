A WordPress plugin to generate content using various AI APIs.

== Description ==

This plugin allows you to generate content (posts or pages) using various AI APIs such as OpenAI, DeepSeek, and Gemini.

== Installation ==

1. Download the plugin
2. Extract the file to the wp-content/plugins folder
3. Activate the plugin in the WordPress dashboard

== Project Structure ==

The plugin is organized as follows:

- /assets
  - /css
    - admin-style.css
  - /js
    - (future JavaScript files)
- /includes
  - /admin
    - admin-pages.php
  - /api
    - api-settings.php
  - /generation
    - content-generation.php
  - /helpers
    - helpers.php
- /languages
  - Translation files
